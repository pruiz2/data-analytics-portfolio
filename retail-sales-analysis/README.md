# Retail Sales Analysis

## Objective
Analyze retail transaction data to identify purchasing patterns across gender, age groups, and product categories.

## Tools Used
- Python
- pandas
- matplotlib

## Key Questions
- Do men and women spend differently across product categories?
- How does spending behavior vary by age group?
- Which categories generate the most revenue?

## Key Insights
- Female customers spend more on Clothing and Beauty products
- Male customers dominate Electronics spending
- Spending patterns vary significantly by age group

- `notebooks/retail_sales_analysis.ipynb` – full analysis
- `data/retail_sales_dataset.csv` – dataset
